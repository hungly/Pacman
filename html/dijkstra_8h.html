Dijkstra algorithm used to determined the shortest path from a node to another node. This algorithm is adapted from the Rosetta's code (2014) page with appropriate modification to accept input provide by my program and provide output in appropriate format to be used by my AI Reference: http://rosettacode.org/wiki/Dijkstra's_algorithm#C
